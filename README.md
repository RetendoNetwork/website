# website

# How to use
## How to start the server
- First, Install [NodeJS](https://nodejs.org) and install the packages `npm install`.
- For start the server, do `npm start`.

## How to git push 
- Put `bash scripts/git.sh`.
- After write a commit and press enter.
- And now it's pushed :^)
